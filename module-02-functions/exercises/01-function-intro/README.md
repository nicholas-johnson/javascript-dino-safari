# Exercise - Introduction to functions

**Mission briefing:** Write your first functions using the `function` keyword. Each one takes parameters and returns a value.

## Tasks

Implement in [`starter/index.js`](starter/index.js):

1. **`add(a, b)`** - returns the sum of `a` and `b`.
2. **`multiply(a, b)`** - returns the product of `a` and `b`.
3. **`greet(name)`** - returns `"Hello, <name>!"`.
4. **`isEven(n)`** - returns `true` if `n` is even, `false` otherwise.
5. **`initials(first, last)`** - returns the first character of each name, uppercased, joined with a dot and trailing dot, e.g. `"E.S."`.
