# Exercise — Map

**Mission briefing:** Use `.map()` to transform array elements into new shapes — numbers, strings, and formatted output.

## Tasks

Implement in [`starter/index.js`](starter/index.js):

1. **`doubleAll(numbers)`** — return a new array with every number doubled.
2. **`extractNames(dinos)`** — given `{ species, zone }[]`, return an array of just the `species` strings.
3. **`formatSightings(dinos)`** — given `{ species, zone, dangerLevel }[]`, return an array of `"species (zone) — danger: N"` strings.
